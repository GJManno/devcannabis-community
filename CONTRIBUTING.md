# How this community works

Welcome to the **DevCannabis** community front porch — the open, low-barrier place to ask questions,
share ideas, and help shape the project.

## You can do this here, no access needed
- **Ask anything** — about the data, a state, the [open spec](https://devcannabis.com/spec), or how to get involved.
- **Propose a state** — tell us which jurisdiction you want covered. Every state starts as a slim statewide "spine" and grows from there.
- **Report a data issue** — spot something off? Flag it with a source. We fix mistakes in the open: https://devcannabis.com/corrections

## Want to contribute data or code?
The data, methodology, sources, and the normalization spec are **open**. The application code is a
**private, invite-reviewed** repo — a vetted-contributor project, not an open-source release. The path
in is open: show up, help out here, and request access at **https://devcannabis.com/contribute** (no
GitHub account needed to start). Ask for the smallest tier that fits — it's easy to grow into the next one.

## Ground rules
Be respectful and constructive — see the [Code of Conduct](CODE_OF_CONDUCT.md). This is a
data-and-tooling community, not a marketplace.
